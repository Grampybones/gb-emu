# gb-emu
Gameboy emulator

Currently playable with most games working. Requires SDL2.dll

Still to do:

* sound and battery support

**Controls**

Start: Enter
Select: RShift
A: Z
B: X
Speedup: Numpad +
Reset: R
Save: S
Load: L

**How to**

drag rom onto exe or run program with rom location as argument
