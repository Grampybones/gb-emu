# gb-emu
Gameboy emulator with a focus on accuracy.

Currently playable with most games working. Uses SDL for video and input so SDL libraries are required.

Still to do:

* sound and battery support
* colour mode

## Controls

Start: Enter  
Select: RShift  
A: Z  
B: X  
Speedup: Numpad +  
Reset: R  
Save: S  
Load: L
