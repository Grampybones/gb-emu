# Gemu
Nintendo Gameboy emulator for Windows.

Currently usable with most games working. Uses SDL for video and input.

Still to do:

* battery support
* colour mode

## Controls

Start: Enter  
Select: RShift  
A: Z  
B: X  
Speedup: Numpad +  
Reset: R  
Save: S  
Load: L
